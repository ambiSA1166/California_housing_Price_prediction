# California Housing Regression

A machine learning project exploring the California Housing dataset using Linear Regression.

## What I Did
- Performed Exploratory Data Analysis (EDA)
- Analyzed feature correlations
- Applied feature scaling using StandardScaler
- Trained a Linear Regression model
- Evaluated model performance using R², MAE, and RMSE
- Conducted residual analysis

## Tech Stack
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn

## Dataset
California Housing Dataset from Scikit-Learn.

## Learning Outcomes
- Data preprocessing and scaling
- Regression modeling
- Model evaluation techniques
- Residual analysis and interpretation
